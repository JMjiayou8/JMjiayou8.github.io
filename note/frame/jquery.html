    # 选择器

    和之前了解过css的选择器可以联系记忆,下面摘选常见选择器

    选择器 说明 备注
    常用
    elem 标签名 $('div')
    #id id属性 $('#box')
    .class class属性 $('.box')
    div,p 所有div元素和p元素 $('div,p')
    div p 选择div元素内的所有p元素 $('div p')
    div>p 选择所有父级是 div 元素的 p 元素 $('div>p')
    div+p 选择所有紧接着div元素之后的p元素 $('div+p')
    div~p 选择div元素之后的每一个p元素 $('div~p')
    属性选择
    [target] 所有带有target属性元素 $('div[target]')
    [target=xxx] 所有使用target="xxx"的元素 $("div[target='xxx']")
    [target!=xxx] 所有使用target不等于"xxx"的元素 $("div[target！='xxx']")
    [target^=xxx] 所有使用target以"xxx"开头的元素 $("div[target^='xxx']")
    [target$=xxx] 所有使用target以"xxx"结尾的元素 $("div[target$='xxx']")
    [target*=xxx] 所有使用target包含"xxx"的元素 $("div[target*='xxx']")
    计数类
    :even
    :odd    		 索引值为偶数/计数的元素
    从 0 开始计数    		 $("tr:even")
    $("tr:odd")
    :eq(i)
    :gt(i)
    :lt(i)    		 匹配一个
    等于/大于/小于给定索引值的元素,
    从 0 开始计数    		 $("tr:eq(1)")
    $("tr:gt(1)")
    $("tr:lt(1)")
    子元素
    :nth-child 匹配其父元素下的第N个子元素
    从 1 开始计数    		 $("ul li:nth-child(2)");
    匹配ul下第2个li元素
    :first-child
    :last-child    		 匹配
    第一个/最后一个元素    		 $("ul li:first-child")
    $("ul li:last-child")
    元素状态
    :hidden
    :visible    		 匹配所有不可见/可见元素 $("tr:hidden")
    $("tr:visible")
    :disabled
    :checked
    :selected
    		 表单元素 $("input:disabled")
    $("input:checked")
    $("select option:selected")
    元素类型
    :input 匹配所有 input, textarea, select 和 button 元素 $(":input")
    :button 所有按钮 $(":button")
    <input type="button" />或
    <button></button>
    :radio
    :checkbox
    :submit
    :reset    		 所有单选按钮
    所有复选框
    所有提交按钮
    所有重置按钮    		 $(":radio")<input type="radio" />
    $(":checkbox")<input type="checkbox" />
    $(":submit")<input type="submit" />
    $(":reset")<input type="reset" />

    # 方法

    摘选常用的方法，更多戳这里~ (opens new window)

    方法 说明 备注
    自定义属性
    attr(name,val) 获取/设置属性 $("img").attr("src");
    $("img").attr("src",'xxx.png');
    removeAttr(name) 删除属性 $("img").removeAttr("src");
    固有属性
    prop(name,val) 获取/设置属性 $("input[type='checkbox']").prop("checked");
    $("input[type='checkbox']").prop("checked", true);
    内容、值
    html(val) 匹配/设置元素的html内容 $('#box').html()
    $('#box').html('html')
    text(val) 匹配/设置元素的内容 $('#box').text()
    $('#box').text('html')
    val(val) 匹配/设置元素的值 $("input").val();
    $("input").val('val');
    class样式类
    hasClass(name) 是否含有某个特定的类 $('#box').hasClass("active")
    addClass(name)
    removeClass(name)    		 新增/删除样式类 $('#box').addClass("active")
    $('#box').removeClass("active")
    toggleClass(name) 如果存在（不存在）就删除（添加）一个类。 $('#box').toggleClass("active")
    筛选
    find() 与指定表达式匹配的元素 $("p").find("span")
    与$("p span")相同
    parent()
    parents()    		 父元素
    prev()
    next()
    siblings()    		 邻居元素
    eq(index) 获取第N个元素,从0开始 $("p").eq(1)
    first()
    last()    		 第一个/最后一个元素 $('li').first()
    $('li').last()
    样式
    css(key,val) 设置样式 $("p").css("color");
    $("p").css("color","#333");
    效果
    show()
    hide()
    toggle()    		 显示、隐藏元素 $("p").show();
    $("p").hide();
    $("p").toggle();
    slideDown()
    slideUp()
    slideToggle()    		 高度变化 $("p").slideDown();
    $("p").slideUp();
    $("p").slideToggle();
    fadeIn()
    fadeOut()
    fadeToggle()
    fadeTo()    		 不透明度变化 $("p").fadeIn();
    $("p").fadeOut();
    $("p").fadeToggle();
    $("p").fadeTo("slow", 0.66);

    # 事件

    事件类型：click,dbclick,hover,change,blur,keydown等等 在选择元素上绑定一个或多个事件的事件处理函数:on(events,[selector],[data],fn)

    $("p").on("click", function(){
  alert( $(this).text() );
});

//也可以直接使用click方法
$("p").click( function(){
  alert( $(this).text() );
});

    模仿悬停事件:hover([over,]out)

    $("p").hover(
  function () {
    $(this).addClass("hover");
  },
  function () {
    $(this).removeClass("hover");
  }
);

checkbox 和 radio 默认选中是蓝色的，但实际运用中，一般都是会根据系统主题色有所变化。如果是现成 UI 库就不用考虑这个问题啦。但涉及到比较旧的系统，引入 UI 库有点不现实，网上有很多解决方法，我就记录一下个人觉得最简便的吧。

checkbox

/*  效果如上所示
  *  一般都是用额外元素遮盖，但个人觉得DOM结构就变复杂啦
  *  用伪类即可实现
*/
input[type='radio'],
input[type='checkbox'] {
  margin-right: 5px;
  cursor: pointer;
  font-size: 14px;
  width: 12px;
  height: 12px;
  position: relative;
  outline: none;
}

input[type='radio']:after {
  position: absolute;
  width: 12px;
  height: 12px;
  top: 0;
  left: 0;
  content: ' ';
  background-color: #fff;
  color: #fff;
  display: inline-block;
  visibility: visible;
  border-radius: 50%;
  border: 1px solid #ccc;
  box-sizing: border-box;
}

input[type='radio']:checked:after {
  border: 2px solid #eb4b4b;
}

input[type='checkbox']:after {
  content: ' ';
  position: absolute;
  width: 12px;
  height: 12px;
  top: 0;
  background-color: #fff;
  color: #fff;
  display: inline-block;
  visibility: visible;
  border-radius: 3px;
  border: 1px solid #ccc;
  box-sizing: border-box;
}
input[type='checkbox']:checked:after {
  content: '✓';
  font-size: 12px;
  line-height: 10px;
  background-color: #eb4b4b;
  border: 1px solid #eb4b4b;
}

关于 select,主要是它难搞的箭头,以及不同浏览器之间的兼容问题。


select {
/_ 自定义边框字体等 _/
width: 160px;
height: 28px;
outline: none;
font-size: 12px;
color: #333;
line-height: 20px;
padding: 4px 10px;
/_如果要加入自定义图片， 就增加这个属性 background: url 或者在 html 中直接加入图标也行_/
background: url('./expand.png') no-repeat 140px center transparent;
background-size: 14px;
/_Chrome 同 Firefox 与 IE 里面的右侧三角显示的样式不同_/
border: solid 1px #ddd;
/_将默认的 select 选择框样式清除_/
appearance: none;
-moz-appearance: none;
-webkit-appearance: none;
padding-right: 14px;
}
/_清除 iIE 的默认选择框样式_/
select::-ms-expand {
display: none;
}
